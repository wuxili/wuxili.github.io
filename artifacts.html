GreenWeb

  • GreenWeb is a set of Web language extensions defined as Cascading Style Sheet (CSS) rules that let developers express user QoS expectations at an abstract level.
  • The Web runtime dynamically determines how to deliver the target QoS experience while minimizing the energy consumption based on programmer-assisted QoS information.

AutoGreen

  • AutoGreen is an automatic annotation framework to annotate (mobile) Web applications with GreenWeb language extensions to enable energy-efficiency optimizations by the Web runtime.
